How It Works

Get up and running in seconds

1

Initialize

Add Hyve to your project with a simple config file

hyve init
2

Create

Spin up an isolated workspace for your feature

hyve create auth server webapp
3

Run

Start your full stack with one command

hyve run auth
4

Cleanup

Remove workspace when done (branches preserved)

hyve cleanup auth
