Workspaces for AI Agents

Run multiple AI agents in parallel on the same codebase. Each gets isolated branches, databases, and ports.

npm install -g hyve-cli üìã
How it works
1
Create isolated workspaces from your main repo
hyve create auth-feature server webapp
2
Each workspace gets its own git branch, database clone, and unique ports
auth-feature
Claude
branch feature/auth
server :4000
database :5500
billing-api
Cursor
branch feature/billing
server :5000
database :5501
ui-refactor
Aider
branch feature/ui
server :6000
database :5502
3
Point any AI agent at a workspace
claude --cwd ~/workspaces/auth-feature
Zero conflicts. Each agent works in complete isolation with its own running services.
